# 🪐 SmartFinance: AI-Integrated Financial Intelligence

**SmartFinance** is a high-density, professional-grade financial management platform designed for users who demand absolute clarity and proactive guidance over their wealth. Built with a "Zero-Scroll" philosophy and integrated with Gemini AI, it transforms raw data into actionable financial wisdom.

![SmartFinance Dashboard](https://images.unsplash.com/photo-1551288049-bebda4e38f71?auto=format&fit=crop&q=80&w=2070)

---

## 🚀 Key Features

### 📊 Advanced Analytics Suite
*   **Dual-View Dashboard**: A high-density, single-viewport interface optimized for 1080p displays. Get a 360-degree view of your finances without ever touching a scrollbar.
*   **Finance Flow**: Real-time visualization of Income vs. Expenses using high-fidelity Area Charts.
*   **Salary Runway**: An innovative declining-balance graph that tracks cumulative spending throughout the month, showing exactly how long your secondary funds will last.

### 🧠 AI Money Coach
*   **Context-Aware Assistant**: A persistent AI chat sidebar integrated with **Google Gemini**.
*   **Proactive Insights**: The coach analyzes your recent transactions to provide personalized saving strategies and spending alerts.
*   **Natural Interaction**: Ask questions like "Can I afford a new laptop this month?" and get data-backed answers.

### 🛠️ Financial Control Tools
*   **Budget Health Gauge**: Instant visual feedback on your current month's spending relative to your income.
*   **Weekly Trend Monitoring**: High-density bar charts tracking spending patterns across different categories.
*   **Local-First Architecture**: Ultra-snappy performance with zero latency, utilizing IndexedDB for secure, local data persistence.

## 💻 Tech Stack

- **Framework**: [Next.js 16 (App Router)](https://nextjs.org/)
- **Visuals**: [Recharts](https://recharts.org/) for high-performance data visualization.
- **Animations**: [Framer Motion](https://www.framer.com/motion/) for premium micro-interactions.
- **Styling**: [Tailwind CSS](https://tailwindcss.com/) with a custom High-Density "True Flat" design system.
- **Database**: [Dexie.js](https://dexie.org/) (IndexedDB wrapper) for robust local storage.
- **AI Engine**: [Google Gemini Pro API](https://ai.google.dev/).

## ⚡ Quick Start

### Prerequisites
- Node.js 18.x or later
- A Google Gemini API Key

### Installation

1. **Clone the repository**
   ```bash
   git clone https://github.com/your-username/integrated-financial-management.git
   cd integrated-financial-management
   ```

2. **Install dependencies**
   ```bash
   npm install
   ```

3. **Set up Environment Variables**
   Create a `.env.local` file in the root directory:
   ```env
   GEMINI_API_KEY=your_actual_api_key_here
   ```

4. **Run the development server**
   ```bash
   npm run dev
   ```
   Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.
